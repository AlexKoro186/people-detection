# People Detection and Counting using YOLO

## Projektbeschreibung

Ziel des Projekts ist die automatische Erkennung, Verfolgung und Zählung von Personen in einem Video.

Zur Personenerkennung werden verschiedene YOLO-Modelle verwendet. Die Personen werden über eine virtuelle Zähllinie gezählt und die Ergebnisse verschiedener Modellvarianten miteinander verglichen.

## Funktionen

- Personenerkennung mit YOLO
- Multi-Object Tracking
- Ein- und Ausgangszählung
- Vergleich mehrerer YOLO-Modelle
- CSV-Export der Ergebnisse
- Erstellung von Diagrammen
- Speicherung annotierter Videos

## Verwendete Modelle

- YOLOv8n
- YOLOv8m
- YOLO11n
- YOLO11s
- YOLO26m
- YOLO26x
